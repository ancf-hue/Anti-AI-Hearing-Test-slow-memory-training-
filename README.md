# 👂 Анти ИИ Проверка слуха
## *медленная тренировка памяти*
### 🎯 О проекте

Интерактивный тренажёр слуха и памяти. Слушайте последовательности цифр, повторяйте их, а на сложном уровне — угадывайте, какую цифру сказал ИИ.

### 📊 Уровни сложности

| Уровень | Шум | ИИ-голос |
|---------|-----|----------|
| 1 | Тишина | ❌ |
| 2 | Средний шум | ❌ |
| 3 | Громкий шум | ✅ (+ угадать ИИ) |

### ✨ Особенности

- 🧠 Адаптивная сложность
- 💾 Сохранение прогресса
- 📱 Адаптивный дизайн
- 🎨 Стеклянный 3D-интерфейс
- 🔴 Яркие подсказки (зелёный/красный)
- 🔊 Реалистичный белый шум
- 🤖 ИИ-искажение голоса
